O que faz um sistema de controle de versão?

  Um sistema de versão permite gerenciar e organizar as diferentes versões (ou uma determinada configuração para um sitema específico) de um software.



Que   benefícios   uma  ferramenta   de  controle   de   versão   pode   trazer   para   uma equipe de desenvolvimento?

  Vários:
  - Desenvolver uma nova versão de um pedaço de software enquanto corrige-se problemas com a versão corrente, de forma paralela
  - Analisar onde uma mudança aconteceu na história do desenvolvimento de um componente
  - Identificar quais versões de código foram incluídas em um componente específico
